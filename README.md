# 幾 A 幾 B

一個使用 React 與 Webpack 製作的「幾 A 幾 B」猜數字遊戲，支援離線單人模式與雙人派對模式。

## 遊戲連結

[開始遊戲](https://kw-yeh.github.io/webpackTest/)

## 功能

- 離線模式：單人猜 4 位不重複數字。
- 派對模式：建立房間或輸入房間碼，和另一位玩家連線對戰。
- 遊戲紀錄：顯示每次猜測的 A/B 結果與最佳步數。
- 行動裝置支援：數字輸入欄位會優先開啟數字鍵盤。

## 本機開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```
